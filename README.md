# PopCatClicker2.0

Готовая HTML5/JavaScript-игра-кликер про POP CAT для публикации на Яндекс Играх.

## Что внутри

- Главный экран с котом, счётом и POP CAT вайбом.
- Магазин улучшений: сила клика, авто-кликер, множитель, временный x2 буст и скины.
- Реалистичные скины на базе исходных картинок проекта с бонусами: новогодний, попкорн и неон.
- POP RUSH: шкала на 1000 кликов, которая включает короткий x3 режим.
- Задания с прогрессом и наградами, чтобы у игрока были цели кроме бесконечного клика.
- Сохранение прогресса в `localStorage` и, при доступности SDK, в Yandex Player Data.
- Поддержка Yandex Games SDK: LoadingAPI, GameplayAPI, interstitial и rewarded video.
- Пауза игры и звука при сворачивании вкладки, потере фокуса и показе рекламы.
- Адаптивный интерфейс для desktop, планшетов и телефонов.

Редактируйте мою игру и меняйте под себя, также пишите что можно добавить в мою игру.
