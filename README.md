# PopCatClicker2.0

Отдельная улучшенная версия PopCat Clicker.

## Что внутри

- Кликер Pop Cat на HTML, CSS и JavaScript.
- Магазин со скинами, x2 множителем и перерождением.
- Музыка, звук клика и регулировка громкости.
- Исправленные русские тексты и пути к ассетам.

## Запуск

Откройте `index.html` в браузере.
